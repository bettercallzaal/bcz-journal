# Ecosystem

A live map of what I'm building and how the pieces connect.

## At a glance

- **The ZAO** — impact network for independent musicians on Farcaster + Base. Member directory, governance, festivals, daily newsletter.
- **WaveWarZ** — onchain music battle platform on Solana. Live battles every weeknight at 8:30 PM EST.
- **ZABAL** — the streaming + coordination layer. Token launch Jan 1, 2026.
- **ZAOOS** — the lab. Where everything new gets prototyped before it earns its own home.

```
                       BetterCallZaal
                       (the connector)
                              │
                              ▼
                            ZABAL
                  (streaming + coordination)
                              │
        ┌─────────────────────┼─────────────────────┐

     The ZAO              WaveWarZ              SongJam

   (the lab)              (live stats)
        │
   graduates ──→  CoC Concertz, ZAOstock, FISHBOWLZ (paused)
```
