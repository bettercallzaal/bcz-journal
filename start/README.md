## The pillars

### 🎵 The ZAO — the impact network

Decentralized hub for independent musicians, artists, and technologists. Members earn **soulbound Respect** for showing up — running events, recording with each other, making the community work. No token sale, no "buy your way in." Build to belong.

[thezao.com](https://www.thezao.com) · [Discord](https://discord.thezao.com) · [`/zao` on Farcaster](https://farcaster.xyz/~/channel/zao) · [@thezaodao](https://x.com/thezaodao) · [@zaofestivals](https://instagram.com/zaofestivals)

### 🌊 WaveWarZ — onchain music battles

Solana-native music battles. Artists go head-to-head; fans trade ephemeral SOL-denominated tokens on outcomes. Live every weeknight at **8:30 PM EST on YouTube**.

[wavewarz.com](https://www.wavewarz.com) · [Live stats](https://wavewarz.info) · [@WaveWarZ](https://x.com/WaveWarZ) · [Mobile app source](https://github.com/bettercallzaal/wavewarzapp) · Co-founder: Ikechi Nwachukwu

### ⚡ ZABAL — the new front-end

The streaming + coordination layer that sits across The ZAO, WaveWarZ, and partner products. Token launches **January 1, 2026**.

[ZABAL Update #3](https://paragraph.com/@thezao/zabal-update-3) (Dec 2, 2025) · Stake $SANG → sqrt multiplier on ZABAL scoring · Long-term vision: **ZAO Stock 2026** physical event in Maine

### 🧪 ZAOOS — the lab

The monorepo where everything new gets prototyped. When something is ready to be a product, it graduates: own repo, own database, own domain, code stripped from ZAOOS so there's no drift. Research stays in ZAOOS forever — it's the institutional memory.

[github.com/bettercallzaal/zaoos](https://github.com/bettercallzaal/zaoos) · [zaoos.com](https://zaoos.com) · [202 research docs](../research/) · Fork it: [FORK.md](https://github.com/bettercallzaal/zaoos/blob/main/FORK.md) + [community.config.ts](https://github.com/bettercallzaal/zaoos/blob/main/community.config.ts)
