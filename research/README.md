# Research

A public surface onto the **[ZAOOS research library](https://github.com/bettercallzaal/zaoos/tree/main/research)** — the institutional memory of everything I've researched while building The ZAO.

**202 active research documents** (plus 79 archived) across 13 topic folders, ~500,000+ words. The library lives in the [zaoos](https://github.com/bettercallzaal/zaoos) repo because that's where the code that consumes it lives. This page is the curated entry point.

## Notes on this surface

- This is a **public mirror** of links. The actual research files live in ZAOOS — that way the docs stay next to the code that uses them, and there's a single source of truth.
- The ZAOOS repo is itself public, so every link above resolves to readable Markdown.
- 42 of the 202 docs are tagged **CANONICAL** — they're living references, kept fresh as the system evolves. The rest are point-in-time research outputs.
- 79 archived docs are preserved at [_archive/](https://github.com/bettercallzaal/zaoos/tree/main/research/_archive) for historical reference.
