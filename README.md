# bcz-journal

A public journal by **Zaal Panthaki** ([@bettercallzaal](https://github.com/bettercallzaal)) — building **The ZAO**, **WaveWarZ**, and the **ZABAL** stack in public.

This repo is the front door. Raw notes, research synthesis, project maps, and what I'm working on, dated. Everything here is public-safe by design — internal-only material lives elsewhere.

## What's here

| Folder | What's in it |
|---|---|
| [nexus/](./nexus/) | **The Nexus** — one door into the whole ecosystem. ZAO, WaveWarZ, ZABAL, ZAOOS. Choose-your-door layout for artists / builders / fans / partners. |
| [journal/](./journal/) | Dated entries. Newest at the top of [`journal/README.md`](./journal/README.md). |
| [research/](./research/) | Public mirror of the [ZAOOS research library](https://github.com/bettercallzaal/zaoos/tree/main/research) — 200+ research docs across music, governance, agents, infrastructure. |
| [ecosystem/](./ecosystem/) | Internal map: every project / repo / on-chain contract / collaborator that makes up the system. More detail than the Nexus. |

## How this is built

Markdown + GitHub Pages. No build step. Drop a file into `journal/`, push, it's live.

Some entries start as raw notes I dump into a working session and then get cleaned up before landing here. The unprocessed versions stay private.
